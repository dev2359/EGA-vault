## 종합
- 미용기기는 본질적으로 일시적·표면 효과 — 심층 침투·근본 개선은 한계. "마사지 효과"라 생각하면 적절.
- LED 마스크가 본 영상 최고 평가 — 게스트 의사 "이거 가지고 있으면 클리닉 안 가도 됨" 호평. 임상 근거 가장 강한 카테고리.
- 침투형 기기 투톱은 파나소닉 (임상 근거·데이터 표기) + 브라이트 (LP 설명력·일렉트로포레이션). 가격은 5.8~5.9만엔 비슷.
- 메디큐브는 풀패키지 (X2) + 가성비 mini 로 라인업 강점이지만 "독자 연구소" 카피는 근거 부족.
- YAMAN 야만은 비타민C 7배 침투 표기로 일본 기기 중 드물게 구체 성분 명시 — 시각적 첫인상 우수하지만 광고 과장 면 있음.
- 쿼드 셀러·글로우 M 크라이오소닉은 차별 기술 (히트쇼크프로테인 / cryogenic) 보유 — 가격 매력 시 고려.
- 플라즈마 샷 부스터는 신기술이지만 일렉트로포레이션 대비 우위 단정 어려움.
- 마이크로커런트 단독 / V 롤러 / 클렌징 브러시는 entry 카테고리 — 효과는 마사지 수준.
- 광고 카피의 "잡티 제거 6.2배" / "크레이터 효과" 같은 표현은 의약외품·과장 영역으로 주의 필요.

## 제품별 평가

### 메디큐브 부스터 Pro X2

*브랜드: [[메디큐브]]*

**총평** — 메디큐브 부스터 Pro 라인 최신 플래그십. 마스크 모드 + 더마샷 X2 + 부스터 + 마이크로커런트 + 에어샷 X2 풀패키지. "전부 다 들어있다"는 의미에서 풀패키지 미용기기로 평가. 단 효과는 일시적 마사지·표면 침투 수준. 헤드가 커서 넓은 부위 사용 편의성 우수. "독자 연구소·박사 설계" 마케팅 카피는 구체 근거 부족. 게스트 의사 평: "기능 자체로는 메디큐브가 패키지로는 가장 우수."

**긍정**
- 풀패키지 — 마스크/더마샷X2/부스터/마이크로커런트/에어샷X2 모드 모두 포함
- 다른 메디큐브 기기 대비 헤드가 커서 넓은 부위 사용 편의성↑
- 브랜드 인지도·디자인 우수

**부정**
- 효과는 일시적 (마사지·표면 침투 위주, 심층 효과 한계)
- "독자 연구소·박사 설계" 마케팅 카피의 구체 근거 부족
- 에어샷 모드는 과거 "쓸모없음" 평가 받았던 기술

**차별점**
- 메디큐브 라인의 풀패키지 플래그십 — Pro / mini 와 차별
- 다양한 모드 동시 운용이 강점 (다용도)

### 브라이트 (BRIGHT) 일렉트로포레이션 미용기기

**총평** — 에가 + 게스트가 파나소닉과 함께 투톱으로 추천한 미용기기, 58,000엔. 일렉트로포레이션 (전기 침투) 기술이 메인 — 피부 자질층 (각질층) 침투 강조. LP 페이지의 피부 메커니즘 설명이 "다른 제품보다 자세하다"고 호평. 결론: "디자인 빼고 기능·효과 비교 시 본인은 브라이트 갈 수도" 라는 게스트 평. 단 임상 근거·데이터 표기는 파나소닉 우위.

**긍정**
- 일렉트로포레이션 기술 — 피부 침투를 메인 기능으로 강조
- LP 페이지의 피부 메커니즘 설명이 친절·자세함
- 58,000엔 — 파나소닉보다 약간 저렴
- 게스트가 "기능 위주면 본인은 이거 갈 수도"라며 강력 후보

**부정**
- 임상 근거·데이터 표기는 파나소닉 대비 약함
- EMS 라인 (엘렉리프트) 사용감은 보통 — EMS 메인 사용은 비추천 (꽤 통증)

**차별점**
- 일렉트로포레이션을 메인으로 내세운 기기
- 파나소닉과 직접 경쟁 — 가격 약간↓ / LP 친절도↑ / 임상 근거↓
- 게스트 매수 후보 1·2위

### 파나소닉 미용기기 (하이퍼 히알루론산 침투 모델)

**총평** — 에가 + 게스트(花선생)가 "이게 진짜 추천"으로 합의한 최종 추천 미용기기. 가격 59,400엔. "히알루론산 12배 / 콜라겐 6.3배 침투" 등 구체 수치 표기 + 자체 실험 데이터 제공. EMS·쿨모드 부가 기능 있으나 핵심은 "EMS·쿨모드는 거의 쓰지 말고 침투 모드만 꾸준히 사용"이 추천 사용법. 임상 근거를 명확히 표기하고 데이터 기반 마케팅으로 신뢰성 우수. 게스트 평: "기본에 충실, 이게 본인이 산다면 산다."

**긍정**
- "히알루론산 12배·콜라겐 6.3배 침투" 등 구체 수치 + 자체 실험 데이터 제공
- 임상 근거를 명확히 표기 — 신뢰성↑
- 게스트 의사가 "이게 진짜 추천"으로 최종 추천
- 꾸준히 쓰기 좋은 기본기에 충실한 설계

**부정**
- 59,400엔 — 가격대 부담
- 브라이트와 비교 시 LP (홈페이지) 설명력은 브라이트가 더 친절
- EMS·쿨모드는 거의 무용 — 핵심 침투 모드만 사용해야

**차별점**
- 임상 근거 + 데이터 표기 측면에서 가장 신뢰성 있는 일본 브랜드 기기
- 브라이트 추격형이지만 데이터 표기로 추월
- 게스트의 최종 픽 — "정직하게 한 대 사겠다면" 1순위

**성분** — 히알루론산, 콜라겐

### YAMAN 야만 포토 플러스 샤이니 (이온부스트 마루치 EX)

**총평** — 야만의 다기능 미용기기, 69,300엔. 비타민C 유도체 침투 "7배" 표기를 가진 유일한 일본 미용기기 (다른 일본 기기는 성분 표기 거의 없음). 에가가 "지금 본 것 중에서 1번 좋아 보인다" 첫인상 평가. 단 본인 실사용 결과 광고만큼의 크레이터·모공 효과는 X (광고 과장). 클리닉 미용기기 베이스 설계로 모드 다양.

**긍정**
- 비타민C 유도체 침투 7배 — 일본 미용기기 중 드물게 구체 성분 표기
- 클리닉 기계 베이스 설계로 모드 다양성·기능성 우수
- 에가가 "지금 본 것 중 가장 좋아 보인다" 첫인상 평가
- 야만 브랜드 — 일본 미용기기 메이커 중 다양한 라인업 보유

**부정**
- 69,300엔 — 본 영상 비교 제품 중 최고가에 가까움
- 실사용 결과 광고만큼의 크레이터·모공 효과는 X (광고 과장)
- 비타민C 유도체 외 다른 성분 표기는 부족

**차별점**

- 클리닉 기계 베이스라 모드·기능 다양함

**성분** — 비타민C 유도체

### 쿼드 셀러 (Quad Cellar)

**총평** — 히트쇼크프로테인 (열로 자가면역 강화) + 초음파 미용기기. 정가 72,000엔, 타임세일 시 37,800엔까지 인하. 화상 환자 75% 회복 연구 논문 인용 — 단, "그 논문이 있다는 것이지 이 기기로 그런 효과가 나는 건 아니다"라고 게스트 의사가 선 그음. 초음파는 미용의료에서도 쓰는 기술이나 출력 수치가 낮음. 글로우 M 크라이오소닉과 같은 가격이면 차가운 쪽을 선택할 듯.

**긍정**
- 히트쇼크프로테인 — 열 기반 자가면역 강화 메커니즘 인용
- 타임세일 시 가격 부담↓ (37,800엔)
- 초음파 + 열 기능 동시 탑재
- 제품 자체로는 나쁘지 않음 ("물건은 나쁘지 않다")

**부정**
- 광고의 75% 회복은 화상 환자 연구 인용 — 이 기기 효과 아님 (오해 소지)
- 초음파 출력 수치가 미용의료 대비 낮음
- "같은 가격이면 차가운 쪽 (글로우 M) 살 듯" 게스트 평

**차별점**
- 히트쇼크프로테인 + 초음파 조합 — 열 기반 메커니즘 특화
- 타임세일 시 가격 매력적

**함량** — 환자 75%, 광고의 75%

**성분** — 환자, 광고의

### 글로우 M 크라이오소닉 프리미엄 쿨링 미용기기

**총평** — 글로우 M 의 차가운 (cryogenic) 기능 탑재 미용기기, 185,900엔. 살롱 동일 30kHz 주파수. "차가운 기능을 가진 미용기기가 드물어서 신기"하다는 게스트 평. 차가운 기능 + 일렉트로포레이션 + 이온 도입 + LED 동시 가능. 단, 18.5만엔 가격은 부담 — "이 돈이면 성형(매몰법)도 할 수 있는 수준" 농담. 비싼 가격이지만 차가운 기능에 가치 부여 시 매력 있음.

**긍정**
- 차가운 (cryogenic) 기능 — 미용기기 중 드문 기능
- 살롱 동일 30kHz 주파수
- 리프트·일렉트로포레이션·이온도입·청 LED 동시 가능 (멀티 기능)
- "같은 가격이면 차가운 쪽 사겠다" 게스트 평 (vs 쿼드 셀러)

**부정**
- 185,900엔 — 본 영상 비교 제품 중 압도적으로 비쌈
- "이 돈이면 미용 시술 가능" 수준의 가격
- 차가운 기능 외 기능은 다른 기기와 차별성 부족

**차별점**
- 차가운 기능 — 미용기기 카테고리에서 흔치 않은 기능
- 멀티 기능 (5+) 동시 탑재로 풀패키지

### 플라즈마 샷 부스터 (Plasma Shot Booster)

**총평** — 저온 플라즈마 + 일렉트로포레이션 결합 신상 미용기기. "이온 유입 600% 초월" 표기 (저온 플라즈마 + 일렉트로포레이션 동시 적용). 플라즈마는 미용의료에서도 사용되며 분자·원자 배열을 일시적으로 흐트러뜨려 성분 침투를 돕는 기술. 게스트 의사 평: "플라즈마가 있어서 끌리지만 다른 기능이 EMS만 있고 부족 — 일렉트로포레이션은 더 우월하므로 굳이 사야 할 이유 부족."

**긍정**
- 저온 플라즈마 — 미용기기에 흔치 않은 기술 (의료용 플라즈마 응용)
- "이온 유입 600% 초월" 표기 — 일렉트로포레이션과 시너지 강조
- 미용의료의 플라즈마 침투 메커니즘 응용

**부정**
- 플라즈마 외 부수 기능 부족 — EMS만 추가
- 동일 가격대에 더 많은 기능 가진 기기 (메디큐브 등) 있음
- "플라즈마가 일렉트로포레이션보다 효과적"이라고 단정 못 함
- 게스트 평: "사고 싶은 이유 부족"

**차별점**
- 저온 플라즈마 기술 탑재 — 신기술이지만 차별점이 약함
- 일렉트로포레이션 + 플라즈마 결합

### LED 마스크 (클리닉 동일 LED 미용기기)

*카테고리: [[마스크]]*

**총평** — 게스트 의사가 가장 강력하게 호평한 미용기기. "클리닉에서 쓰는 LED 와 동일하다" 평가 + "이거 정도면 (클리닉 매월 갈 필요 없이) 이거 가지고 있으면 됨"이라는 결론. 빨강 + 다양한 색 LED 20분 / 10분 사용. 모드별 추천 미용 성분 표기 (콜라겐·레티놀·시카·플라센타·비타민 등) → 게스트가 "납득감 있다"고 평가. 임상 근거 측면에서도 LED 가 미용기기 중 가장 강함 (염증 진정, 콜라겐 생성, 항균 등 보고).

**긍정**
- "클리닉 LED 와 동일" — 게스트 의사 최고 호평
- "이거 가지고 있으면 클리닉 안 가도 됨" 수준 평가
- LED 임상 근거 — 미용기기 중 가장 강함 (염증·콜라겐·항균)
- 모드별 추천 미용 성분 표기 → 사용 가이드 명확
- 빛 기반이라 다른 침투 기기처럼 피부 부담 X

**부정**
- 사용 시간 (20분) 다소 길어 매일 루틴 불편 가능
- 제품명 본문 명시 안 됨 (구체 모델명 확인 필요)

**차별점**
- 게스트 의사 최강 추천 — 본 영상 최고 평가 제품
- LED 기반 — 다른 침투형 미용기기와 카테고리 다름
- 클리닉 동일 사양 + 가정용 가격

### 마이크로커런트 미용기기 (단독 모델)

**총평** — 마이크로커런트 단독 미용기기, 40,000엔 (타임세일 시 29,000엔). 미세 전류로 세포 활성화·신진대사 촉진 표방. 게스트 의사 평: "임상 근거는 약하고 마사지 효과 정도로 생각하는 게 적절." 미용의료에는 없는 기능 (=미용기기 전용). 입문용 가격대로 "한 번 해보고 싶은 사람"에게 추천.

**긍정**
- 40,000엔 (세일 시 29,000엔) — 미용기기 입문 가격대
- 마사지 효과 (혈류 개선·붓기 완화) 측면에서 일정 효과
- 미세 전류 — 피부 부담 적음

**부정**
- 임상 근거 약함 — 마사지 효과 정도로 한정
- 미용의료에는 없는 기능 (=세포 활성화는 과학적 근거 부족)
- 단독 기능이라 메디큐브 같은 풀패키지와 비교 시 가성비↓

**차별점**
- 마이크로커런트 단독 — 단순·저렴
- 미용기기 카테고리 entry 가격대

### V 롤러 미용기기 (Vローラー 30kHz 진동)

**총평** — V 롤러 형태 미용기기, 30kHz 주파수 진동. 1초당 약 2만 회 진동으로 마사지 효과. 게스트 의사 평: "이 시린 사람 (치아 통증 있는 사람) 에게 좋을 수 있다" — 진동이 다른 통증을 분산시키는 효과. EMS 단독으로 피부 탄력·턱 라인 개선은 "거의 무리." 일시적 슈팅 효과만 가능.

**긍정**
- 30kHz 진동 — 마사지 효과 확실
- 이 시린 사람에게 부수적 효과 가능
- 일시적 슈팅·붓기 완화 가능

**부정**
- EMS 단독으로는 피부 탄력 개선 불가 — 일시적 효과만
- 턱·목 처짐은 EMS 만으로는 개선 어려움 (게스트 의사 명시)
- 근본적 효과 측면에서 미용의료와 격차 큼

**차별점**
- V 롤러 형태 — 마사지 + EMS 단순 결합
- 일시적 부기·붓기 완화 특화

### 모공 케어 클렌징 브러시 (Mini + 브러시 라인)

*카테고리: [[클렌징]]*

**총평** — 모공·클렌징 케어 브러시 기기, 18,000엔. 솔 굵기 0.6~1.1mm. 게스트 의사 평: "이 가격이면 시중 더 저렴한 클렌징 기기 많고, 이걸 사기보단 그냥 세수가 낫다." 모공 안쪽 청소가 필요할 만큼 더러운 피부는 아니라는 지적 — "미용기기 살 정도 의식 높은 사람의 피부는 이미 깨끗하다." 진동 + 마사지로 모공 주변 이완 효과는 인정.

**긍정**
- 진동 + 마사지로 모공 주변 이완
- 솔 0.6~1.1mm 두께로 피부 자극 적음 ("계란 안 깨지는 부드러움")
- 클렌징 + 마사지 동시 가능

**부정**
- 18,000엔 — 시중 더 저렴한 클렌징 기기 많음
- 게스트 평: "이 돈 쓸 거면 그냥 세수가 낫다"
- 모공 안쪽 청소는 과잉 — 미용기기 살 정도 사람은 이미 피부 깨끗
- 과도한 사용 시 오히려 피지 분비 증가 가능 (역효과)

**차별점**
- 클렌징 + 마사지 특화 — 다른 침투형 미용기기와 카테고리 다름
- 모공 케어 entry 가격대
