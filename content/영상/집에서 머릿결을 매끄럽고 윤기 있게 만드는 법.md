# 집에서 머릿결을 매끄럽고 윤기 있게 만드는 법.

#주제/피지 #브랜드/다이소 #브랜드/러쉬

**한 줄 요약** — 집에서 머리카락을 부드럽고 윤기 있게 만드는 방법

## 종합
- 두피와 머리카락의 기름과 냄새 제거가 중요함
- 열을 이용한 헤어 케어로 윤기와 보습을 강화할 수 있음
- 과도한 제품 사용은 머리카락 손상을 초래할 수 있으므로 주의가 필요함
- 가벼운 사용감의 트리트먼트가 선호됨

## 제품별 평가

### 러쉬 빅

*브랜드: [[러쉬]]*

**총평** — 두피의 기름과 냄새 제거에 효과적임. 두피 관리가 필요한 사람에게 적합함

**긍정**
- 두피의 기름과 냄새를 효과적으로 제거함
- 사용 후 두피가 상쾌해지고 좋은 향이 남음

**부정**
- 사용 후 머리카락이 뻣뻣해질 수 있음

**차별점**
- 두피의 기름과 냄새 제거에 특화된 소금 스크럽

---

### 다이소 코믹스

*브랜드: [[다이소]]*

**총평** — 열을 이용한 윤기 강화에 효과적임. 머리카락에 윤기를 더하고 싶은 사람에게 적합함

**긍정**
- 열을 이용해 머리카락을 보수하며 윤기를 더함
- 무겁지 않은 마무리감 제공

**부정**
- 과도한 사용 시 머리카락 손상 가능성 있음

**차별점**
- 열을 이용한 보수와 윤기 강화 기능
