*카테고리: [[선크림]], [[크림]], [[클렌징]]*

**총평** — 일반 피부 타입에게 적합하며, 세정력과 향이 개선된 점에서 추천할 만함

**긍정**
- 부드러운 거품과 뛰어난 세정력으로 피부가 매끄럽고 깨끗해짐
- 향이 개선되어 상쾌한 느낌을 줌
- 사용 후 건조함이 덜 느껴짐

**부정**
- 민감한 피부에는 알칼리성으로 인해 자극을 줄 수 있음
- 여드름이 잘 발생하는 피부에는 적합하지 않음

### 아넷사

*카테고리: [[선크림]], [[크림]]*

**총평** — 焼けにくさ와 落ちにくさ 측면에서 압도적 1위로 평가. 사용감은 ビオレ 아스리즘 대비 다소 떨어지지만 메이크업 효과보다 차단력을 우선시할 때 아넷사를 선택. 산·해변·여행 갈 때 본인 default 픽.

**긍정**
- 焼けにくさ + 落ちにくさ 둘 다 圧倒的 1위
- 海·山·野外 활동 시 default 픽
- 깨진 자외선 차단 막이 재형성되는 형상 기억 기술 탑재 (연구의 대화 영상 참조)

**부정**
- 사용감은 韓国 코스메 대비 떨어지는 편 (다소 무겁고 건조감)
- 가격이 2,500~3,000円 정도로 ビオレ 대비 비쌈

**차별점**
- 使用感 < 焼けにくさ 우선 user 의 절대적 1픽
- 映画館 데이트 같은 실내 상황엔 굳이 아넷사 아니어도 OK — 상황별 사용 분리

### 퍼펙트휩 프리미엄 각질 케어

*카테고리: [[선크림]], [[크림]]*

**총평** — 薬部外品으로 サリチル酸 함유. 여드름 殺菌·예방 + 각질 케어 + 透明感 컨셉이지만 본인은 그 좋다는 평가가 잘 와닿지 않음. 본인의 1군 픽은 일반 퍼펙트휩 水色 버전.

**긍정**
- サリチル酸 함유로 여드름 殺菌·예방 기능
- 약용 화장품 (薬部外品) 분류
- 각질 케어 + 透明感 효과 主張

**부정**
- 巷의 칭찬과 달리 본인 사용감으로는 차이 잘 모르겠음
- 본인은 일반 水色 버전이 더 만족

**차별점**
- 퍼펙트휩 라인 내 각질 케어 특화 라인 (水色 일반 버전과 분리)
