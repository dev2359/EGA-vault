# LED 마스크 (클리닉 동일 LED 미용기기)



효능: [[주름-탄력]], [[진정]] · 카테고리: [[마스크]]


> 1개 영상에서 다룸. 봇이 신규 학습 시 갱신.


## 에가의 평가

### [【Qoo10 메가 세일】미용기기, 도대체 의미가 있나? "전문가의 시선으로 구매한" 이 제품이다‼︎](https://www.notion.so/Qoo10-36e7aa4b7b0f81c984dbd3917d674856)  ([YouTube](https://youtu.be/Xoo9FnAQqo8))

**총평** — 게스트 의사가 가장 강력하게 호평한 미용기기. "클리닉에서 쓰는 LED 와 동일하다" 평가 + "이거 정도면 (클리닉 매월 갈 필요 없이) 이거 가지고 있으면 됨"이라는 결론. 빨강 + 다양한 색 LED 20분 / 10분 사용. 모드별 추천 미용 성분 표기 (콜라겐·레티놀·시카·플라센타·비타민 등) → 게스트가 "납득감 있다"고 평가. 임상 근거 측면에서도 LED 가 미용기기 중 가장 강함 (염증 진정, 콜라겐 생성, 항균 등 보고).

**긍정**
- "클리닉 LED 와 동일" — 게스트 의사 최고 호평
- "이거 가지고 있으면 클리닉 안 가도 됨" 수준 평가
- LED 임상 근거 — 미용기기 중 가장 강함 (염증·콜라겐·항균)
- 모드별 추천 미용 성분 표기 → 사용 가이드 명확
- 빛 기반이라 다른 침투 기기처럼 피부 부담 X

**부정**
- 사용 시간 (20분) 다소 길어 매일 루틴 불편 가능
- 제품명 본문 명시 안 됨 (구체 모델명 확인 필요)

**차별점**
- 게스트 의사 최강 추천 — 본 영상 최고 평가 제품
- LED 기반 — 다른 침투형 미용기기와 카테고리 다름
- 클리닉 동일 사양 + 가정용 가격
