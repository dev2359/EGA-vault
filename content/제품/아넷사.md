# 아넷사



효능: [[자외선차단]] · 카테고리: [[선크림]], [[크림]]


> 1개 영상에서 다룸. 봇이 신규 학습 시 갱신.


## 에가의 평가

### [【선크림・세안】역시 이거 사게 되.. 돈키호테 대량 구매.](https://www.notion.so/3647aa4b7b0f81b8b88fe071b79c4094)  ([YouTube](https://youtu.be/j9vdGiGMYWU))

**총평** — 잘 타지 않는 차단력과 잘 지워지지 않는 지속력 측면에서 압도적 1위. 사용감은 비오레 아스리즘보다 다소 떨어지지만, 메이크업 효과보다 차단력을 우선할 때는 아넷사를 선택. 산·해변·여행 갈 때 본인의 기본 픽.

**긍정**
- 차단력(잘 안 탐)과 지속력(잘 안 지워짐) 둘 다 압도적 1위
- 바다·산·야외 활동 시 기본 픽
- 땀·마찰로 무너진 자외선 차단 막이 다시 형성되는 형상 기억 기술 탑재

**부정**
- 사용감은 한국 코스메보다 떨어지는 편(다소 무겁고 건조함)
- 가격이 2,500~3,000엔 정도로 비오레보다 비쌈

**차별점**
- 사용감보다 차단력을 우선하는 사용자에게 절대적 1순위
- 영화관 데이트 같은 실내 상황에선 굳이 아넷사가 아니어도 OK — 상황별로 구분해 사용
